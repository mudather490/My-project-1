🎮 Hangman Game - Python Beginner Project
A simple Hangman game built with Python where the player tries to guess a hidden word before running out of lives.
Features:
🎲 Random word selection
🔤 Letter guessing system
❤️ Limited lives
📝 Tracks used letters
🏆 Score calculation
🔁 Play again option
🧩 Organized with Python functions
